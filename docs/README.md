# Clinic Management System Documentation

Welcome to the documentation for the Clinic Management System. This documentation provides an overview of the system's modules, data models, and architecture.

## Role-Based Modules

The system is organized into modules based on user roles:

- [Admin Modules](modules/admin.md): Comprehensive management of clinic operations, staff, patients, and system settings.
- [Doctor Modules](modules/doctor.md): Tools for doctors to manage appointments, patient records, and schedules.
- [Staff Modules](modules/staff.md): Functionalities for clinic staff to handle appointments, attendance, and patient registration.
- [Patient Modules](modules/patient.md): Access for patients to view their records and appointments.

## System Overview

- [Data Model](data_model.md): Detailed information about the core data entities and their relationships.
- [Architecture](architecture.md): High-level overview of the system architecture, security, and RBAC.
